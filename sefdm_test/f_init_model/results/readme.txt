Имя файла:
[алгоритм_демодуляции]_[алгоритм_детектора]_[кол-во_поднесущих]_[альфа]_[модуляция]_[@параметр_алгоритма_детектора].mat

Пример 1:
        MF_ZF_16_0.8_BPSK.mat
        
        MF - алгоритм демодуляции
        ZF - алгоритм детектора
        16 - кол-во поднесущих
        0.8 - альфа
        BPSK - модуляция
        
        @параметр_алгоритма_детектора - отсутствует
        
        
Пример 2:
        MF_ID_26_0.72222_BPSK_3.mat
        
        MF - алгоритм демодуляции
        ID - алгоритм детектора
        26 - кол-во поднесущих
        0.72222 - альфа
        BPSK - модуляция
        3 - кол-во итераций в алгоритме ID-детектора == @параметр_алгоритма_детектора
